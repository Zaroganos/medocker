Medocker Standalone Executables
===============================

This package contains standalone executables for Medocker:

- medocker: Main CLI application with all commands
- medocker-configure: Configuration tool
- medocker-web: Web-based configuration interface

Getting Started:
1. Run 'medocker' to see available commands
2. Run 'medocker-configure --interactive' to set up your configuration
3. Run 'medocker-web' to start the web interface
4. Run 'medocker deploy' to deploy your Docker stack

For more information, visit: https://github.com/Zaroganos/medocker
